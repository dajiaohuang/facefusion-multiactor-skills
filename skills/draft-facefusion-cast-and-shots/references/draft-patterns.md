# Draft Patterns

Use this phase when the project still needs its first valid draft state.

Phase goals:

- establish source candidates
- establish shot boundaries
- establish enough structure for later reference review

Ask for:

- source face files
- target video or videos
- rough shot ranges

Do:

- create draft `cast.json`
- create draft `shots.json`
- prefer incomplete but usable draft state over waiting for perfect detail

Do not:

- force final role merge decisions
- force optional shot pipeline choices
- force preview approval or job execution
